Incitatus-OS
============

Stage 3 Dissertation project. The aim is to create an operating system as a vehicle to learn about operating systems from a student perspective.

##Objectives

	› Modular
		› Orthogonal
		› Dependency control
		› Module Interfaces
	› Extensible
		› Driver interface
		› Allow user-space development (through a minimal C library)
	› Unambigous code design
		› Simplicity over complexity
		› Object oriented design
		› Unit testing
	› Clear and concise documentation
		› Focus on 'Why?' more than 'What?'
